# Cocktail Sort
Cocktail Sort is a variation of Bubble sort.
The Key Differences between the Bubble  and cocktail sort algorithms, is that the Bubble sort always traverses data from left and moves the largest element to its correct position in the first sorting and second-largest in the second and so on. Cocktail Sort traverses through a given array in both directions alternatively, which Cocktail sort does not need to go through the unnecessary iteration making it efficient for large arrays.
## Function
The code uses a predetermined array and sorts it with cocktail sort.

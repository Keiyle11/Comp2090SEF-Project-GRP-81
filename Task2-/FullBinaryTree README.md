# Full Binary Tree
The Binary Tree is a data structure where each node can contain a value and a maximum of two children. These children would also have a maximum of two children. This data structure is efficient in data organisation, sorting, searching and more. However in this particular type of binary tree, the strict binary tree only allows a node to have 0 or 2 children.

## Sorting Algorithms Related to Binary Trees
Here are some of the sorting algorithms that navigate through the binary tree.

Pre-order starts from the root node and chooses the left path until there is none. After that, it would start going through the right nodes starting from the root note. 

In-order starts from the left-most node and works its way up to the root node, not including the right nodes of the tree. After that, it starts from the root node and goes through the right nodes from there.

Post-order goes through a binary tree's left subtree starting from the left-most node, then goes through the right subtree starting from the right-most node. Lastly, the order would list the root node.
